# mmplugin-channel-recommender

When user join new channel, this plugin recommends other channels to the user like `Users who joined this channel also join {{.Channels}}.`.

![](./screenshot.png)

# Usage
1. Download `org.kaakaa.mmplugin-channel-recommender-*.*.*.tar.gz` from [Releases](https://github.com/kaakaa/mmplugin-channel-recommender/releases)
2. Upload downloaded file in mattermost plugin system_console (**System Console** > **PLUGINS (BETA)** > **Management**)
